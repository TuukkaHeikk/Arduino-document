# Arduino Dashboard (React + Vite + Chart.js)
This project is a ready-to-run React dashboard for Arduino Cloud data.

## Quickstart
1. Extract the zip.
2. Install Node.js 22+.
3. In the project folder run:
   ```
   npm install
   npm run dev
   ```
4. Open http://localhost:5173

## Security
- DO NOT place your Arduino Cloud API token in frontend code.
- Create a small backend proxy (example in `server-proxy-example`).
